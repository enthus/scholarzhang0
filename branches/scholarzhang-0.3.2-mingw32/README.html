张秀才 0.3.2

安装

  • 解压
  • 下载安装WinPcap 4.1 beta5
  • 创建快捷方式或者在命令行运行scholarzhang.exe

运行

  • 创建快捷方式或者在命令行运行scholarzhang.exe，需要一个参数。
  • 首次运行，参数设为减号"-"以显示网卡列表，记录所需网卡，形如\Device\NPF_{01234567-89AB-CDEF-FEDC-BA9876543210}
  • 再次运行，参数设为\Device\NPF_{01234567-89AB-CDEF-FEDC-BA9876543210}选取网卡。注意cmd.exe的命令行参数要求在\Device\NPF_{A4467544-11E3-42D4-8011-3FFAE3A2FF47}两边加上单引号'以避免\被转义。

BUGS

  • 某些正常站点的时候正常通信在打开张秀才的时候可能被张秀才误重置连接。
